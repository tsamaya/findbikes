Trouver des vélos

est une application pour trouver une station pour prendre ou rendre des vélos JCDecaux, dans les villes équipées.

Cette application utilise l'API OpenData de JCDecaux et l'API ArcGIS for Javascript d'ESRI.

Pour trouver une station de vélo, sélectionner une ville de la liste, puis zoomer sur votre emplacement ou utiliser la localisation de votre équipement mobile.

Vous avez aussi la possiblité de saisir une adresse pour vous localiser sur la carte.
